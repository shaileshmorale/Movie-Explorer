# Movie-Explorer

## Project Description

The Movie Explorer Web App is a responsive web application that allows users to search for movies and explore movie details using a public API. The application demonstrates the use of JavaScript, API integration, and responsive UI development.

Users will be able to search for movies, filter results, sort data, and interact with the interface dynamically.

## Purpose

The purpose of this project is to demonstrate knowledge of:

* JavaScript programming
* Fetch API integration
* Array Higher-Order Functions
* Responsive web design
* Dynamic user interface development

## Public API Used

OMDb API (Open Movie Database)

This API provides movie data such as:

* Movie title
* Year
* Poster
* Rating
* Genre
* Plot summary

## Planned Features

* Search movies by title
* Filter movies by year
* Sort movies alphabetically
* Display movie posters
* Responsive design for mobile, tablet, and desktop
* Loading indicator while fetching data
* Error handling for invalid searches
* Save favorite movies using local storage
* Dark mode / light mode toggle
